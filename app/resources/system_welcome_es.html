Ahora, usted puede iniciar la creación de las clases en la estructura del template:

  1. Cree la clase de control. Ex: ProductForm, ProductList;
  2. Inicie sesión como administrador (administrador) para registrar los permisos;
  3. Registre la clase como programa (Administración -> Programas). Debe rellenar el campo "clase de control" con el nombre de la clase (por ejemplo, ProductForm);
  4. Conceda permisos a la clase registrada para algún usuario o grupo (Administración -> Usuarios o Administración -> Grupos);
  5. Agregue el programa al menú .xml, que contiene la estructura del menú;
  6. Realiza un logout y un login con algún usuario con permiso para ejecutar el programa creado;
  7. Finaliza la creación del programa.
